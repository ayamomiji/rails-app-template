= Rails App Template

Inspired by https://github.com/ihower/rails3-app-template

== Usage

  git clone git://github.com/ayamomiji/rails-app-template.git
  rails new your_app_name -TJ -d postgresql -m rails-app-template/rails4.rb

This will setup some gems and create db `your_app_name`.
